# Amalgam

Common re-usable module repository for the convenience of Android application development.

## Download

Grab via Gradle

```groovy
repositories {
    mavenCentral()
    maven { url 'https://raw.github.com/nohana/Amalgam/master/amalgam/repository/' }
}
android {
    dependencies {
        compile 'com.amalgam:Amalgam:0.2.1'
    }
}
```
